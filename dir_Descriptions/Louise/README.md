# How to call Louise

## Call
```
with Louise ;
```

## Usage
### fnUpperAll()
```
result := Louise.fnUpperAll( "pRiNcEsSeS" ) ;
-- result = "PRINCESSES"
```

### fnLowerAll()
```
result := Louise.fnLowerAll( "pRiNcEsSeS" ) ;
-- result = "princesses"
```

### fnUpperFirst()
```
result := Louise.fnUpperFirst( "pRiNcEsSeS" ) ;
-- result = "Princesses"
```
